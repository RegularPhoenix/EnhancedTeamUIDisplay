Enhanced Team UI Display (ETUD) allows you to see HP and some other stats of another players on your team (Up to 3 players, additional panels can be enabled in config). UI is toggled by pressing 'L' by default.

Compatible with Calamity Mod

Feel free to give your feedback on mod on Steam Workshop page.
If you found a bug, please report it to RegularPhoenix#2044

Update 1.1.0:
	- Mod is now client-side (This means that this mod will not be disabled client-side if the server does not have it.)
	- Added boss summary option
	- Added extended ally info option
	- Fixed (hopefully) UI position not saving on exit
